This HTML document links all the documents (in EARMARK format) and results of the tests described in the article Recognising document components in XML-based academic articles, submitted for publication to DocEng 2013.

  • Gold standard (in XML formats) made for comparing the outcomes of the algorithm: Balisage set.

  • Archives containing the documents (stored as EARMARK files) used in the tests: Balisage set.

  • Archives containing the assignments of DoCO classes to each document of both the test sets (stored as XML and HTML files): Balisage set.

  • Statistics of the outcomes of the algorithm, calculated for each document in both sets: Balisage statistics (HTML and XML).
