This HTML document links all the documents used in the evaluation part of the submitted JASIST article A Semantic Web Approach To Everyday Overlapping Markup.

Each following section includes a table describing seven different versions of a wordprocessor document in three different formats: the OpenOffice Writer format (ODT), the Microsoft Word format (DOCX) and EARMARK linearised in Turtle. Each version of the former two formats is accompanied by the respective XML source containing the textual content of the version itself.

All the document versions are downloadable as archive too.
